Projeto Website para Divulgação de Currículo

Objetivo
Desenvolvimento de um website (html, css, imagens) contendo os currículos do grupo e a implantação de um ou mais serviços na nuvem AWS, com rede de entrega de conteúdo, com baixa latência, com segurança, e armazenamento durável, para atender a demanda de 11 milhões de requisições do tipo get e select por mês.
Tecnologias
AWS S3 Standard

  • Amazon Simple Storage Service (AWS S3), é um serviço de armazenamento virtual ilimitado de objetos, compartilhamento de arquivos ou criação de Website estático, repositório de data lakes e análise de big data oferecendo baixa latência e alta velocidade, com durabilidade de 99,999999999 % (onze noves) e disponibilidade de 99,99%, com sete classes de armazenamento

  • Será utilizado o serviço AWS S3 Standard para armazenamento de objetos, com políticas de restrição de acesso apenas pelo endereço do site

AWS Route53

  • O serviço AWS Route53 será utilizado para resolução do domínio e criação das
    zonas de dns necessárias para realizar o acesso seguro.

AWS Certificate Manager

  • Este serviço será usado para gerar um certificado ssl para o site, permitindo o
    uso de segurança ssl para o domínio.

AWS CloudFront

  • Este serviço será utilizado para permitir a integração do certificado ssl ao domínio,
    restringindo o acesso somente pelo site e não diretamente pelo bucket S3.

  • Também ajustará as políticas de acesso, permitindo acesso global com rede de
    entrega de contúdo e com baixa latência como solicitado nos objetivos.
